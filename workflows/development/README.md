# Development Workflows

Workflows for code analysis, refactoring, and development best practices.

## Available Workflows

| Workflow | Description |
|----------|-------------|
| [api-compatibility-check](./api-compatibility-check.ds.md) | Check if an API supports a specific method or capability |
| [architecture-patterns](./architecture-patterns.ds.md) | Identify inconsistent patterns and architectural drift |
| [config-schema-check](./config-schema-check.ds.md) | Check if a config system supports a specific option |
| [dependency-capability-check](./dependency-capability-check.ds.md) | Check if dependencies support a required feature |
| [test-coverage-gap-analysis](./test-coverage-gap-analysis.ds.md) | Find untested code and test quality issues |

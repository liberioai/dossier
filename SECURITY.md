# Security Policy

## Reporting Vulnerabilities

**Do not open public GitHub issues for security vulnerabilities.**

Report security issues via GitHub's private vulnerability reporting:
https://github.com/liberioai/dossier/security/advisories/new

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

- Initial response: within 48 hours
- Status update: within 7 days

## Scope

In scope:
- MCP server vulnerabilities
- Workflow validation bypasses
- Information disclosure

Out of scope:
- Social engineering
- Issues in third-party dependencies
- Vulnerabilities in user's execution environment
